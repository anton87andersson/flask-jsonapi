# Flask JSON API
An easy to use and sample JSON API with Flask

## How to use

First step
```p
First start flask develop-server
```

```c
python3 application.py
```


Get the data
```p
Go to http://127.0.0.1:5000/api/v1/resources/programming/all
```

or run the code

```c
python3 get_json.py
```


Post data

```c
# Run the post.py file
python3 post.py
```
